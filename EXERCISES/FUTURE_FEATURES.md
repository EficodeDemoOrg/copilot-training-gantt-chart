## Ideas fo future features

### Smaller changes
- Task reordering — drag rows vertically to reorder (position already exists in the schema but isn't editable from the UI).
- Add milestones to the charts
- Change the color of the task


### Bigger features
- Labels can be added to the tasks for filtering.
- Multiple charts
- Dependencies / predecessors — link tasks (finish-to-start), render arrows, prevent invalid drops, and auto-shift dependents when a predecessor moves.
